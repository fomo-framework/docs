Introduction

Fomo is a web application framework based.

We tried to implement FOMO in the simplest possible way so that anyone can use it.

Fomo supports the following:

  • Simple, fast routing engine.
  • Processing work in the background
  • Queued job processing,and....

Fomo is very fast, simple and for use in large scales (Of course, it still has flaws grin)

First Fomo Project

tip

We have not run FOMO on operating systems (Windows) so far, and you may encounter problems installing FOMO on these two operating systems. (If there is a problem, send us the error message)

Before creating your first Fomo project, you should ensure that your local machine has PHP and Composer installed.


After you have installed PHP and Composer, you may create a new Fomo project via the Composer create-project command: 

                                    composer create-project fomo/fomo example

After creating the project, start the Fomo local development server using Mr. Engineer: 

                                    cd example

php engineer server:start

Once you have started the development server by Mr. Engineer, your application will be accessible in your web browser at http://localhost:9000.

Mr. Engineer

In Fomo, the boss is Mr. Engineer. It means that all orders are issued by Mr. Engineer

Commands like:

  • Start the HTTP server
  • Start the queue server
  • Start the scheduling server
  • Making controller, resource, middleware, service, job, task and...
  • Run tests
  • And more...

To see the commands that Mr. Engineer can issue You can run the following command 

                                    php engineer
